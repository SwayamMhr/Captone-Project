<p>

<p>
  This project is a high performance MERN stack backend designed to streamline two-wheeler rentals through a secure API and robust data management.
  In this project I worked as the backend developerusing "Node.js" and "Express.js" for this Capstone project.
</p>


<h3> Backend Tech Stack</h3>
<ul>
  <li><b>Runtime:</b> Node.js</li>
  <li><b>Framework:</b> Express.js</li>
  <li><b>Database:</b> MongoDB with Mongoose</li>
  <li><b>Security:</b> JWT (Authentication) & Bcrypt (Password Hashing)</li>
  <li><b>Validation:</b> Yup</li>
</ul>


<h3> File Structure Highlights</h3>
<ul>
  <li><code>index.js</code>: Primary server entry point and database connection logic.</li>
  <li><code>/user</code>: Account-related controllers and database models.</li>
  <li><code>/product</code>: Core logic for inventory management and vehicle availability.</li>
  <li><code>/middleware</code>: Security layers including authentication guards and ID validators.</li>
</ul>


<h3>Quick Start</h3>
<ol>
  <li>Navigate to the directory: <code>cd backend</code></li>
  <li>Install all dependencies: <code>npm install</code></li>
  <li>Configure environment: Setup <code>.env</code> with your <code>MONGO_URL</code> and <code>SECRET_KEY</code></li>
  <li>Launch the server: <code>npm run dev</code></li>
</ol>
